# Dataset

The H&M Personalized Fashion Recommendations dataset is **not committed** to git (too large; gitignored).

## Download

1. Create a free Kaggle account: https://www.kaggle.com
2. Visit the competition page and accept the rules:
   https://www.kaggle.com/competitions/h-and-m-personalized-fashion-recommendations/data
3. Download these three files into this folder:
   - `articles.csv` — product catalogue (~105K items)
   - `customers.csv` — customer profiles
   - `transactions_train.csv` — purchase history (~31M rows)
